# Android Projects

A beginner-friendly Android application built using Java and Android Studio to practice Android app development fundamentals, UI design, and Gradle-based project architecture.

---

## 🚀 Overview

This project was created to learn the fundamentals of native Android development using Java and Android Studio.

The application demonstrates:

- Android project structure
- Activity lifecycle basics
- XML-based UI design
- Material Design themes
- Gradle build system setup
- Android testing structure

---

## ✨ Features

- Native Android application using Java
- Material Design 3 theme integration
- Responsive XML layouts
- Edge-to-edge UI support
- AndroidX libraries integration
- Unit and instrumentation testing setup
- Gradle Kotlin DSL configuration
